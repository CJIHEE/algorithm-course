## ❗ 풀이 방법
1. 처음 시작점에서 상,하,좌,우 탐색하여 갈수 있는 길 탐색
2. 갈 수 있는 길을 찾았을때 cont를 바로 올려주지 않고 배열을 이용해 현재 있는 칸에서 +1 해서 한칸 옮길 때마다 +1해주는 방법으로 카운트 해야한다
3. 각 칸수에는 이동하기 전 칸 수에서 +1 한 값이 담기게 된다
