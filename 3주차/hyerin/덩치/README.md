## ❗ 풀이 방법
1. 입력받은 사람의 weight, height, rank에 대한 정보를 담은 내부 class를 만들어 문제를 해결했다.
2. 입력받은 사람과 다음 사음사람의 weight와 height를 비교한 뒤 등수를 조정하는 완전탐색 사용

## 🙂 새로 알게된 점

* 내부 클래스로 만들어서 사용하니까 어떤 사람인지 구분이 확실하게 가서 편리하긴 했는데, 그냥 배열로 받아서 해결했어도 문제없었을 것 같다.
